安装

需要GitHub Copilot 订阅，学生免费，但验证学生身份比较麻烦，我虽然通过了，但主邮箱没有收到成功邮件。

Copilot Chat还在waiting list，等通过了再更新

在Visual Studio Code的扩展中安装GitHub Copilot并登录

基本操作

要接受建议，请按 Tab。

拒绝所有建议，请按 Esc。

OS 查看下一个建议 查看上一个建议
Windows Alt+] Alt+[

  1. 你可能不想获得 GitHub Copilot 提供的任何初始建议。 可以使用键盘快捷方式来提示 GitHub Copilot 在新选项卡中显示多个建议。

    若要打开具有多个其他选项的新选项卡，请按 Ctrl+Enter。

  2. 从注释生成代码建议

    可以在注释中使用自然语言描述要执行的操作，GitHub Copilot 会提供代码建议来实现你的目标。

还可以使 GitHub Copilot 为 API 和框架生成建议。 以下示例使用 GitHub Copilot 创建将返回当前时间的简单 Express 服务器。

function calculateDaysBetweenDates(begin, end) {
const millisecondsPerDay = 1000 * 60 * 60 * 24;
const millisBetween = end - begin;
const days = millisBetween / millisecondsPerDay;
return days;
}
