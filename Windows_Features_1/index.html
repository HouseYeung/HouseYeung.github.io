Windows被忽视的实用功能（一）

发表于 2023-07-05 分类于 Tutorials 阅读次数： 本文字数： 1.2k
让你的电脑更好用


视频超分辨率

NVIDIA新出的AI功能，用RTX显卡提升视频的分辨率和画质，在720P以下的低分率视频上有明显效果，但也会增加视频播放时显卡的占用率。

安装步骤

  1. 确保电脑是NVIDIA的RTX30或40系显卡，更新到最新显卡驱动。
  2. 在桌面右键，打开NVIDIA控制面板，在视频选项的调整视频图像设置中启用超分辨率。这里有4档质量可选，我用4060显卡开质量4看视频时显卡占用约40%。推荐开到4，效果最明显。
  3. 在edge中启用：用edge访问edge://flags/#edge-video-super-resolution，并开启该选项即可。
  4. 在potplayer中启用：打开播放器，右键视频-视频输出设置，把视频渲染器改成内置direct3D 11视频渲染器，在下面开启D3D11 GPU超分辨率。

运行Android应用

随着移动设备的普及，许多互联网公司开始更加重视移动端应用而非桌面端应用，部分公司甚至完全取消了桌面应用的开发。过去运行Android应用主要靠模拟器，模拟器功能完善且自定义程度高，但启动加载慢，不少还带有广告。而Windows 11自带的安卓子系统和Windows浑然一体，点击应用图标即可直接启动，十分流畅。我认为模拟器更适合玩Android游戏，而安卓子系统更适合运行应用程序。

安装步骤

  1. 首先确保你的系统是Windows 11，且运行在固态硬盘上。把国家或地区改成美国，并在控制面板的window功能里开启虚拟化。完成后重启电脑

  2. 点击下载 Amazon AppStore，安装时会自动下载并安装Windows Subsystem for Android（WSA），完后不用管Amazon AppStore了，建议直接卸载。打开WSA，打开设置里的开发者模式。

  3. 点击下载apk安装程序，之后双击apk安装包就可以安装Android程序了。当然也可以用adb命令行安装。

mod应用

Android有比Windows更多的魔改版软件，可以去掉广告，甚至解锁订阅功能。下面是我用过的mod应用网站，注意不是上面所有的应用都管用，一般来说流媒体类的应用是无法破解的，许多比较小众的应用也不一定能找到。

https://liteapks.com/
https://apkdone.com/
https://happymod.com/

Android模拟器

之前试过的里比较推荐夜神模拟器，功能最全，就是系统自带一些植入广告。

实时字幕

Windows自带的字幕功能，可以自动识别电脑音频并生成对应的字幕。目前测试下来准确度并没有很高，即便是英语也会有不少错误，还要改进。好处是完全在本地运行，无需联网。

安装步骤

  1. 确保你的系统是Windows 11，在设置的辅助功能中开启字幕。或者使用win+Ctrl+L快捷键。
  2. 识别特定语言需要在系统的语言和区域设置中下载对应语言的语音识别包
