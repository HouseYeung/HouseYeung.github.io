在Mac OS运行Windows游戏

发表于 2024-01-25 分类于 Tutorials 阅读次数： 本文字数： 2.7k
ARM也要跑x86！


工具选择

目前据我所知有三种主流解决方案：apple自己出的Game Porting Toolkit（简称GPT或GPTK），开源的Whisky，收费的crossover。整个过程基本和使用steam deck的兼容层一致，下面逐一介绍：

CrossOver是一个付费软件，使用了Wine来在mac上兼容Win应用程序，但是不支持dx12游戏，需要打cxpatcher补丁增加GPT

Game Porting Toolkit游戏移植工具包于2023年6月6日在WWDC大会上发布，将Wine与苹果自家的D3DMetal技术相结合，实现对DirectX 11和12的支持。核心其实就是核心是 CodeWeavers 公司开发的软件 CrossOver

Whisky差不多是增加了UI的套壳GPT，能帮你更方便的配置环境

除此之外还有Herioc Games Launcher，这个第三方启动器能调用上述工具的Wine来运行epic和gog商店中的游戏，有易用的UI和全面的配置项，是运行这两个平台游戏的最佳方案。

总结一下，推荐使用CrossOver+cxpatcher或Whisky二选一，千万不要单独使用GPTK，过于繁琐了。

GPTK安装和配置

下面是我安装GPTK的过程和不建议使用GPTK的具体原因，推荐直接跳到whisky的部分

首先要安装Rosetta转译

softwareupdate --install-rosetta
进入x86模式的终端，注意之后每次重启终端后都要执行该命令，包括启动游戏前

arch -x86_64 zsh

安装x86版本的Homebrew

/bin/bash -c "$(curl -fsSL https://gitee.com/ineo6/homebrew-install/raw/master/install.sh)"

brew环境自动切换，先进入配置文件

nano ~/.zshrc

粘贴下面内容

按下 Ctrl+ X，然后按Y（表示“是”）来保存更改，并且按 Enter键退出。

重新加载

source ~/.zshrc

接下来要编译GPTK，非常耗时且不必要，我大概跑了一个小时，期间第一次听到macbook的风扇

在Apple开发者下载网站下载Game_porting_toolkit，挂载后执行，注意下载的版本区别

配置Wine prefix

把系统版本改成win10

接下来就可以尝试安装steam了

虽然大概率是进不去的，还要复制config、userdata、registry.vdf三个文件到下面这个目录

~/my-game-prefix/drive_c/Program Files (x86)/Steam

加载运行没有性能显示框，强烈建议关闭

gameportingtoolkit-no-hud ~/my-game-prefix 'C:\Program Files (x86)\Steam\xxxx.exe'

再遇到的bug就没有仔细研究了，主要还是这种终端启动的方式太麻烦了，推荐直接安装GPT的教程都是在耍流氓

Whisky安装和配置

只需要
brew install --cask whisky
启动后按照屏幕上的说明操作

点击右上角的+新建Bottles，这相当于一个小的Windows系统，且可以自定义版本，推荐使用win10 19042

按下右下角的 Run…按钮，导航到您在 Finder 中下载.msi或 .exe文件的位置，选择文件并按 Open

一般来说不需要建立多个bottles，虽然在一个地方安装太多游戏会使依赖性混乱。根据我在steamdeck的经验，只要能运行就不需要新建

常见问题

  • 无法加载防作弊：没错，包括PUBG、Fortnite、Apex Legends、Valorant等带有防作弊的游戏理论上在Wine不起作用。（实际上steamdeck能运行apex，Easy Anti-Cheat 和 BattleEye应该能运行，但我没有测试，据我所知在Linux上运行apex有极大概率误封）

  • 无法运行DX9游戏：Whisky 专注于使用 DX11 或 12 的现代游戏，在这种情况下，CrossOver 是更好的选择

  • 无法运行Epic商店：请使用Herioc Games Launcher，没有比它更好的选择，在官网Download Heroic下载mac版本，它会自动检测本地已经安装的Wine

Whisky还是CrossOver?

由于我没有测试CrossOver，只能在下面简单比较

Feature 特征 Whisky CrossOver
完全开源 ✅ ❌
永久免费更新 ✅ ❌
MSync ✅ ✅
ESync ✅ ✅
DirectX 12 支持 ✅ ✅
自动安装 ❌ ✅
Denuvo支持 ❌ ✅
EA 应用程序支持 ❌ ✅
Wine 8 支持 ❌ ✅
技术支持 ❌ ✅

看起来CrossOver支持一键安装，兼容更多app，尤其是要运行EA或者D加密游戏时，但是Whisky它免费啊！
