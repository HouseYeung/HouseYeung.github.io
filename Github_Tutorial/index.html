Git和Github的简单教程


Git

github是一个基于git的远程代码托管平台，所以要用github我们首先要安装git。其实github也可以在网页上直接上传文件，但操作非常的不方便，还有许多限制。而借助git就方便很多。

git是一个版本管理工具，可以在电脑上创建你文件的历史版本创库，这样你就可以在手滑的时候进行版本还原，或者进行一些奇怪修改的时候创建分支。
