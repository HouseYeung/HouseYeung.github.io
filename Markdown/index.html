一、概述

1.1 常见误解纠正

  • Markdown 发布的成品是完全由标点符号标签组成的纯文本。
  • Markdown 不限制编辑器，只是一种文本格式。

1.2 Markdown编辑器

  • 目前我在使用Typora和Gridea做Markdown编辑器

1.3 可扩展 HTML 语法

  • 对于 Markdown 中未包含的标签, 可以直接使用 HTML标签，例如用 HTML 标签替代 Markdown 的链接语法。
